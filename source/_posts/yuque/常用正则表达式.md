## 1.将数字中间几位以\*号代替，前后各保留 4 位

```javascript
var string = "1234567894";
var s = string.replace(/^(.{4})(?:\d+)(.{4})$/, "$1***$2");
console.log(s); // 1234***7894
```

## 2.字符串驼峰转下划线

```javascript
var string = 'isGood'
var s = string.replace(/([A-Z])/g, '_$1').toLowerCase()
console..log(s) // is_good
```
